생성자
	
	1. 생성자는 클래스 이름과 같다
	2. 생성자는 리턴형이 없다. 필드(객체) 초기화가 목적임
	3. 매개변수를 갖는다, 오버로드가 가능하다 (매개변수가 있어서 중복 가능)
	4. 객체 생성시 호출된다
	5. 생성자를 넣어주지 않으면, default 생성자가 자동으로 생성
		new Test01();
		
	6. 매개변수가 있는 생성자를 넣어주면, default 생성자는 생성되지 않음
	      필요시 프로그래머가 default 생성자를 넣어주어야 한다
	
	Test01.java
	
	this.전역변수=매개변수; //매개변수 값을 전역변수에 할당한다
	

	
	7. this : 현재 클래스 객체
	   this() : 자신의 클래스 안의 생성자 호출, 거의 사용하지 않음
	   this(매개변수) 
	   
	   this()는 생성자 안에서 첫줄에 기재해야 한다
	   
	   super : 상위 클래스 객체를 말한다  (사용하지 말것)
	   super() : 상위 클래스 생성자 호출 (자주 사용)
	  
	■  super()는 생성자 안에서 첫줄에 기재해야 한다
	
          ※     this()와 super()는 동시에 사용 불가
